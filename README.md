# Gulliver UNIVPM – Portale Ufficiale 🔴⚓

![Gulliver Logo](/public/logo-gulliver-tondo-png.png)

Benvenuti nel repository ufficiale di **Gulliver – Lista di Rappresentanza UNIVPM**. Dal 1987, siamo il punto di riferimento per la comunità studentesca dell'Università Politecnica delle Marche, difendendo i diritti e promuovendo la cultura in tutto l'Ateneo.

Questo portale è progettato per essere lo strumento digitale principe per lз studentз, offrendo informazioni, supporto e partecipazione attiva alla vita universitaria.

---

## 🚀 Funzionalità Principali

Il portale raccoglie diversi strumenti e aree tematiche:

- **🗳️ Elezioni Studentesche**: Una sezione dedicata alla campagna elettorale, con i programmi completi, l'elenco di tuttɜ i nostri candidatɜ e il manifesto della lista.
- **📚 Kit dellə Studentə**: Una guida essenziale e scaricabile per orientarsi tra tasse, scadenze, servizi dell'Ateneo e vita nella città di Ancona.
- **💬 Gruppi Matricole**: Raccolta verificata di tutti i link ai gruppi WhatsApp e Telegram per ogni corso di laurea UNIVPM.
- **🎸 Gulliver Rock**: Sezione dedicata allo storico festival musicale di Ancona, con la programmazione dell'edizione corrente e l'archivio delle passate.
- **🎮 Gulliver46**: Un'esperienza interattiva e ludica dedicata alla campagna elettorale.
- **🤝 Rappresentanza & Convenzioni**: Tutte le informazioni sul nostro lavoro negli organi accademici e le agevolazioni attive per lз studentз.

---

## 🛠️ Tech Stack

Il progetto è costruito con le tecnologie web più moderne per garantire prestazioni, accessibilità e facilità di manutenzione:

- **Framework**: [Next.js 15](https://nextjs.org/) (App Router)
- **Linguaggio**: [TypeScript](https://www.typescriptlang.org/)
- **Styling**: Vanilla CSS con Custom Properties (per un design system fluido e coerente)
- **Hosting**: Netlify
- **Design System**: Ispirato ai principi della modernità e del premium design, con forti richiami all'identità visiva del Gulliver.

## 🤝 Open Source & Contribuzione

Crediamo fermamente nel valore della condivisione. Questo progetto è **Open Source** perché vogliamo che sia un'opera collettiva della comunità studentesca. 

Se sei uno studente appassionato di coding o vuoi semplicemente aiutare a migliorare i contenuti:
1. Fai un **Fork** del repository.
2. Crea un **Branch** per la tua funzionalità (`git checkout -b feature/nuova-idea`).
3. Effettua un **Commit** delle tue modifiche.
4. Apri una **Pull Request**!

Ogni contributo, dalla correzione di un typo all'aggiunta di una nuova sezione, è prezioso.

## 💻 Sviluppo Locale

Se vuoi avviare il progetto sul tuo computer:

1. Clona il repo: `git clone https://github.com/lorenzocalifano/gweb2026.git`
2. Installa le dipendenze: `npm install`
3. Avvia il server di sviluppo: `npm run dev`
4. Apri [http://localhost:3000](http://localhost:3000)
